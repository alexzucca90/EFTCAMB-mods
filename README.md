# EFTCAMB-mods
A few mods for EFTCAMB that I need for my projects.

## Generalized-Brans-Dicke project

### Installation

Put the files:
- GBD_utils.f90                             -> in /eftcamb/
- 04p9_interpolated_parametrizations_1D.f90  -> in /eftcamb/04f_parametrizations_1D/

Run 
  - make clean
  - make eftcamb_dep
  - make eftcamb
